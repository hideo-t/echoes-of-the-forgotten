# echoes-of-the-forgotten

「世界王 - Echoes of the Forgotten」は、19歳の怖がりな少女リコが主人公の、100章からなる壮大な物語体験アプリです。雨降る街で見つけたカセットテープをきっかけに、完璧ではない世界の美しさと、誰もが自分の小さな世界の王になれることを描いた、感動的なインタラクティブノベルです。

**特徴:**
- 100章の完全な物語（省略なし）
- 4つの分岐点での選択システム
- レスポンシブデザイン（スマートフォン最適化）
- 美しいイラストと音楽的な世界観
- 「未完でもいい」というメッセージ

**技術仕様:**
- 単一HTMLファイル（CSS/JavaScript内包）
- GitHub Pages対応
- 画像フォールバック機能
- プログレッシブな進行システム

物語は完璧を求めない世界の在り方を探求し、読者自身が「王」になることで終わります。

## 技術スタック
- HTML5 / CSS3 / Vanilla JS
- Mobile-first responsive design
- Hosted on GitHub Pages
